## Web Dev Portfolio using Vue.js
There are two branches on this repo, each containing a different Vue project.

### On 'main' branch:
"project-one" is a simple random quote generator where the user
can save quotes.

### On 'products' branch:
"project-two" is a simple product viewing website that is inspired
by the local sari-sari stores found in the Philippines.

## To Run
git clone git@github.com:corpuzJai/ProjectVue.git \
npm install \
npm run dev
